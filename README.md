## Instruções para rodar a aplicação

1. Clone o repositório de back-end https://github.com/Aquinolima/api-dioshopping
2. npm i
3. npm run dev

4. Clone o repositório de front-end https://github.com/Aquinolima/dioshopping
5. npm i
6. npm start
